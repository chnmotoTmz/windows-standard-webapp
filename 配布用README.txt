本管理アプリケーション
==================

このアプリケーションはWindowsの標準機能だけで動作する、シンプルな本管理Webアプリケーションです。
特別なソフトのインストールは不要で、手持ちの本のコレクション管理に役立ちます。

【主な機能】
・本の登録、一覧表示、検索、編集、削除
・JSONデータのダウンロード、アップロード、編集
・ブラウザベースのシンプルなインターフェース

【起動方法】
1. run-server.bat をダブルクリック
   ※管理者権限が必要な場合があります
2. PowerShellの画面が表示され、自動的にブラウザが開きます
3. アプリケーションが表示されたら、使用開始できます
4. 終了するときは、PowerShellの画面を閉じてください

【注意事項】
・開発中のアプリケーションのため、不具合がある可能性があります
・データは data\books.json に保存されます
・画像のアップロードには対応していません

【教育利用について】
このアプリケーションは、中学生向けのプログラミング教育教材としても利用できます。
以下の教育用資料が含まれています：

1. 要件定義書_中学生向け.md - 簡単な言葉で書かれた機能説明
2. 動作フロー図.md - アプリの動作の流れを図解
3. 講習用スライド.md - 授業で使えるスライド形式の教材（実習課題付き）

プログラミング初心者でも理解しやすい構成になっていますので、教育現場でご活用ください。
アプリを改造する実習なども行うことができます。

【使い方】

2. 自動的にブラウザが開き、アプリケーションが起動します
3. アプリケーションを終了する場合は、コマンドプロンプトウィンドウを閉じてください

【注意事項】
・Windows 10以降のOSで動作します
・管理者権限は不要です
・追加のソフトウェアインストールは不要です
・ポート8080が使用されていないことを確認してください

【フォルダ構成】
このフォルダ内のファイルは削除しないでください：
・start.bat - アプリケーション起動用
・server-debug.ps1 - サーバープログラム
・public/ - Webアプリケーションのファイル
・data/ - データ保存用フォルダ

【トラブルシューティング】
Q: アプリケーションが起動しない
A: 以下の点を確認してください：
   1. ポート8080が他のアプリケーションで使用されていないか
   2. PowerShellが利用可能か
   3. アンチウイルスソフトがブロックしていないか

Q: データが保存されない
A: data/フォルダに書き込み権限があるか確認してください
